<h1>Madlibs Code </h1>
 ```console
name = input("Enter name: ")
adjective_1 = input("Enter a adjective: ")
fruit = input("Enter a fruit: ")
color = input("Enter a color: ")
adjective_2 = input("Enter a adjective: ")
adjective_3 = input("Enter a adjective: ")
verb = input("Enter a verb ending with ing: ")

print(f"There is a boy named {name} who lived in a small village. ")
print(f"{name} had a {adjective_1} farm.")
print(f"{name} likes to grow {fruit} in his {adjective_1} farm.")
print(f"What was special about {name}'s {fruit} is because of their {color} color.")
print(f"The {color} {fruit} are {adjective_2}.")
print(f"{name} was {adjective_3} and {verb}")
 
 ```
