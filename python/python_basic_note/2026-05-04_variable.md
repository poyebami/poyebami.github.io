<h1>Variable</h1>
Variables is a container for a value (string, integer, float, boolean). A variable behaves as if it was the value it contains. Each variable should have a unique name. A variable is a reusable container for a value.  
 ```console
 $ # example of variable with unqiue names
 
 $ # create a variable named 'age' and assign it to the number 25
  age = 25

 $ # create a variable named 'name' and assign it to a string
  name = "prosper"

 # Use the variable with f-string
 print(f"{name} is {age} years old.")
 ```
<h3>Strings</h3>
A `string` is a series of text or characters. It can be used with `""` or `''`. They can include numbers but we treat them as charachers.
 ```console
 first_name = "prosper"
 ```
 The variable `first_name` will behave as if it was the value in "". So in this case, it is `Prosper`.

```console
$ # example of strings
first_name = "prosper"
food = "sushi"
email = "sushi@email.com"

print(f"Hello {first_name}")
print(f"You like {food}")
print(f"Your email is: {email}")

Hello prosper
You like sushi
Your email is: sushi@email.com
```
<h3>Integers</h3>
A integer is a whole number. Integers should not be in quotes `""` because it would be a `string`
 ```console
 $ # integer
 age = 25

 $ # not a integer
 age = "25"
 ```
Another example of integer could be a quantity.
 ```console
  quantity = 3

  print(f"You are buying {quantity} items")
 ```
```console
 $ # examples of integers
 age = 25
 quantity = 3
 num_of_students = 30

 print(f"You are {age} years old")
 print(f"You are buying {quantity} items")
 print(f"Your class has {num_of_students} students")

 You are 25 years old
 You are buying 3 items
 Your class has 30 students
```
<h3>Floats</h3>
Floats means floating point number. A float is a number but contains a decimal portion. 
 ```console
 price = 10.99

 print(f"The price is ${price}")
 ```
```console
 $ # example of floats
 price = 10.99
 gpa = 3.2
 distance = 5.5

 print(f"The price is ${price}")
 print(f"Your GPA is {gpa}")
 print(f"You ran {distance}km")

 The price is $10.99
 Your GPA is 3.2
 You ran 5.5km
```
<h3>Boolean</h3>
A boolean is either true or false. When T/F value for boolean makes sure that first letter is capital. With boolean values, you don't output them directly. You likely to see them internally within a program, such as when workign with `if statement`.
 ```console
 is_student = True
 
 print(f"Are you a student?: {is_student}")
 Are you a student: True
 ```
```console
$ # example of booleans
is_student = False
for_sale = True
is_online = False
print(f"Are you a student?: {is_student}")

if is_student:
    print("You are a student")
else:
    print("You are not a student")

if for_sale:
    print("This car is for sale")
else:
    print("This car is not for sale")

if is_online:
    print("prosper is online")
else:
    print("prosper is not online")

You are not a student
This car is for sale
prosper is not online
```

<h3>F-String</h3>
You can use your variable with some text by using `F-String`. `F-String` (formatted string literals) being a string with `f` or `F` before the opening quotation mark or triple quotation mark. Inside the string , you can write a Python expression between { and } characters that can refer to variables or literal values. 
 ```console
 name = "Alice"
 age = 30
 print(f"Hello, {name}. You are {age} years old.") 

 Hello, Alice. You are 30 years old.
 ```
The `f` means format. To use our variable in f-string. The variable will have to be inside curly braces `{}`.
 ```console
 print(f"Hello {name}")
 ```
